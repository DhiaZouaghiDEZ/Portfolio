# Software Engineer Portfolio

A responsive React application for a software engineer portfolio website.

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Hero section with call-to-action
- About section with skills
- Projects showcase
- Contact form
- Clean, modern styling

## Development

This project uses [Vite](https://vitejs.dev/) with React and TypeScript.

### Development server

Run `npm run dev` for a dev server. Navigate to the URL shown in the terminal. The application will automatically reload if you change any source files.

### Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/` directory.

### Running unit tests

Run `npm test` to execute tests using [Vitest](https://vitest.dev/).

## Customization

- Edit `src/App.tsx` to change content
- Modify `src/index.css` for styling changes
- Update `src/main.tsx` for new entry behavior
